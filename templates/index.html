Dark Pattern Identification

Working Process

  1. Data Collection and Preparation: We gather data from both normal websites and those known to employ dark patterns. This data is then preprocessed to remove null values and duplicate entries, ensuring quality input for our models.
  2. Model Training: Two machine learning models are trained separately for presence detection and pattern categorization. The presence detection model employs a pipeline consisting of TF-IDF vectorization and a Bernoulli Naive Bayes classifier, while the pattern categorization model utilizes TF-IDF vectorization and a Multinomial Naive Bayes classifier.
  3. Scraping and Analysis: When a user submits a website URL, our system scrapes the website to extract its text content. The text content is then processed using the trained models for dark pattern detection. If a line is predicted to contain a dark pattern, the model for pattern categorization further predicts the category of the dark pattern.
